數學期望值
=============

.. note::
    試求亞洲父母對他們小孩成績的期望值(大誤)

期望值
------------

今天園遊會，你們班要賣福袋來 :strike:`斂財` 賺錢，總共有分100, 500, 1000 元的福袋，各有5, 3, 2 包，一個賣350元，那他的平均價值有多少

.. math::
    \frac{100\times 5 + 500\times 3 + 1000\times 2}{10} = 400

.. note::
    有沒有覺得他長得很像加權平均值，我也覺得很像

對於這種東西，我們給他一個名字叫做 `E` ，他的定義長這樣

.. math::
    E = \sum^{n}_{i=1} m_i\ p_i

其中

.. code-block:: text

    E = 平均價值
    n = 事件的數量
    Mi 第i個事件的價值
    Pi 第i個事件的機率

當期望值遇上組合
-------------------

有位 :strike:`白痴` (喂) 客人去買你們的福袋，假設他花了700元，那他的所得金額期望值是多少

總共有2000,1500, 1100, 1000, 600, 200 元 的這些可能性

然後再用機率算出來

.. math::
    2000 \times \frac{C^2_2}{C^{10}_2} + 1500 \times \frac{C^2_1 C^3_1}{C^{10}_2} + 1100 \times \frac{C^2_1 C^5_1}{C^{10}_2} + 1000 \times \frac{C^3_2}{C^{10}_2} + 600 \times \frac{C^3_1 C^5_1}{C^{10}_2} + 200 \times \frac{C^5_2}{C^{10}_2} = 800

然後再算數的你覺的這種題目太 :strike:`雞掰` 麻煩了，所以你想到另外一個簡單的方法

今天他抽兩次 :math:`\rightarrow` 他每個東西都乘兩倍，所以他也可以算成這樣

.. math:: 
    \frac{100\times 10 + 500\times 6 + 1000\times 4}{10} = 800

.. tip::
    對這種東西也可以用平均值來解，是不是看起來沒有這麼抽象了

小練習
---------

園遊會時間
+++++++++++

今天學校要辦園遊會，聰明的你決定要向班上提出一番賞的 :strike:`斂財計畫` 商品提案，同學們也覺得非常的不錯，於是就在園遊會時販賣

商品有分為

.. code-block:: text:

    超大福袋    2包   500元等值商品
    大福袋     10包   100元等值商品
    中福袋     15包   80塊等值商品
    小福袋     20包   50塊等值商品
    這是什麼啦  30包   10元等值商品

請問你們班成功 :strike:`斂財` 獲利的最低門檻的定價是多少

.. tip::
    可以用平均價值比較好理解

.. math::
    500\times \frac{2}{77} + 100\times \frac{10}{77} + 80\times \frac{15}{77} + 50\times \frac{20}{77} + 10\times \frac{30}{77}

經過精密的計算，我們會得到結果為 :math:`58.\overline{441558}` 約為59元

一直吃麥當勞
++++++++++++++++

題天有十個漢堡，有兩個麥香雞與八個麥香魚，請問隨便拿三個拿到麥香雞的期望值

今天有三種情況，拿到兩個，一個與零個，再套到機率

.. math::
    2\times \frac{C^2_2C^8_1}{C^{10}_{3}} + 1\times \frac{C^2_1C^8_2}{C^{10}_3} + 0\times \frac{C^8_3}{C^{10}_3}

其中前面的 :math:`C^2_n` 就是麥香雞的組合，經過精密的計算得出 :math:`\frac{3}{5}` 個

或是你可以這樣想

今天拿到麥香雞的機率是 :math:`\frac{1}{5}` ，那拿三次的機率就是 :math:`\frac{3}{5}` 是不是非常的美妙

賭博時間(?)
---------------

今天有一個遊戲，他要同時丟三個硬幣，如果出現一個正面拿到5元，兩個正面可以得到10元，三個正面可以得到15元，如果沒有正面的話，就要給莊家100元。
那玩一次遊戲的期望值有多少，假設今天再改一次規則，如果沒有正面的話，要付給莊家60元，那他的所得期望值是多少。

1. 丟三個硬幣時，他有3,2,1,0個正面的情況，可以數字化變成
   
   .. math::
        5\times \frac{C^3_1}{2^3} + 10\times \frac{C^3_2}{2^3} + 15\times \frac{C^3_2}{2^3} + (-100)\times \frac{1}{2^3}
    
經過精密的計算過後，得到期望值為-5元

2. 如果在全無正面的時候，要付給莊家60元，那期望值為
   
   .. math::
        5\times \frac{C^3_1}{2^3} + 10\times \frac{C^3_2}{2^3} + 15\times \frac{C^3_2}{2^3} + (-60)\times \frac{1}{2^3}

得到期望值為0元

.. tip::
    當一個遊戲的期望值為0元時，我們稱這個遊戲是**公平的**
